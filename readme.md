- Vue3

- Tailwindcss
  

## :dart:  Goals
- Discover Vue.js version 3

- Explore Tailwindcss
- UX / UI

## :link: How to use

### :gear: Launch the development server & meeting on http://localhost:3000/

### Project setup
```javascript
npm install
```

### Compiles for development
```javascript
npm run dev 
or 
npx vite
```

### Compiles and minifies for production
```javascript
npm run build
or 
npx vite build
```
